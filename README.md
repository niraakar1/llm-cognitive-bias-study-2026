# Do Large Language Models Think Like Us? The Scaling Law of Rationality

This repository contains the full automated data pipeline, statistical graphing tools, and raw 1,350-trial dataset for our behavioral replication study measuring systematic cognitive biases in LLMs.

Our findings indicate a **"Scaling Law of Rationality"**—smaller models successfully replicate human flaws like the Sunk Cost Fallacy, while highly-scaled models (Llama 70B, Command R+, GPT-4o-mini) shed human heuristics entirely, acting as sterile, expected-value optimizers.

### 🚀 The "Two-Hour Agentic Publication"
This entire pipeline—from multi-threaded API querying to statistical analysis, visualization mapping, and manuscript drafting—was fully autonomously coded and generated by an AI Agent working with a non-technical enthusiast over a two-hour period. We are publishing the raw Python engine and CSV data to maintain absolute open-science transparency regarding AI-collaborative academic workflows.

## Repository Structure
- `run_experiments.py`: The core multi-threaded Python engine. Connects to Google, Groq, Cohere, and OpenAI APIs with exponential backoffs.
- `analyze_results.py`: The Seaborn/Matplotlib visualization script. Automates bootstrapped 95% Confidence Intervals, Cohen's *d* effect size generation, and *t-test* analysis.
- `results/`: Contains `raw_responses.csv`, the actual raw dataset consisting of N=1,350 randomized, logically-parsed responses.
- `figures/`: High-resolution, publication-grade visualization PNGs.
- `paper_outline.md`: The full academic manuscript detailing the methodology, behavioral heuristics, and discussion.

## Measured Paradigms
We evaluated 6 modern LLMs (`Command R`, `Command R+`, `Llama 3.1 8B`, `Llama 3.3 70B`, `GPT-4o-mini`, `Gemini 2.5 Flash`) across 5 classical behavioral paradigms:
1. **Anchoring Effect**
2. **Framing Effect (Asian Disease Problem)**
3. **Sunk Cost Fallacy**
4. **Base Rate Neglect**
5. **The Decoy Effect**

## Setup & Reproduction
1. Clone this repository.
2. Install pip dependencies: `pip install requests pandas numpy scipy matplotlib seaborn python-dotenv`
3. Add your API keys to an `.env` file (refer to the script for required keys).
4. Run `python run_experiments.py`.
